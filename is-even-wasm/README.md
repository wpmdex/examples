# is-even-wasm

Rust → multiple binding targets example. Checks if a number is even or odd.
Demonstrates `wasm-bindgen`, `cbindgen`, and `pyo3` — each target gets its own type declarations.

## Structure

```
is-even-wasm/
├── Cargo.toml              # Rust package config
├── build.rs                # Runs cbindgen to generate C header
├── src/
│   ├── lib.rs              # Core logic + module declarations
│   ├── wasm_bindings.rs    # wasm-bindgen (compiled only on wasm32)
│   ├── ffi.rs              # C FFI (excluded on wasm32)
│   └── python_bindings.rs  # PyO3 (opt-in via --features python)
├── include/
│   └── is_even.h           # Generated C header (cbindgen)
├── python/
│   └── is_even_wasm.pyi    # Python type stubs
├── demo.html               # Browser demo
└── pkg/                    # Generated by wasm-pack (do not edit)
    ├── is_even_wasm.d.ts   # TypeScript declarations
    ├── is_even_wasm.js     # JS glue (ESM)
    └── is_even_wasm_bg.wasm
```

## Prerequisites

- [Rust](https://rustup.rs/)
- [wasm-pack](https://rustwasm.github.io/wasm-pack/installer/) — for WASM target
- [maturin](https://www.maturin.rs/) — for Python target

## Build targets

### WASM + TypeScript (`.d.ts`)

```sh
wasm-pack build --target web
```

Output in `pkg/`. Generates `is_even_wasm.d.ts` automatically from `#[wasm_bindgen]` annotations.

Usage:
```ts
import init, { is_even, is_odd } from "./pkg/is_even_wasm.js";

await init();

console.log(is_even(4)); // true
console.log(is_odd(3));  // true
```

### C (`.h`)

```sh
cargo build
```

`build.rs` runs `cbindgen` on `src/ffi.rs` and writes `include/is_even.h`.

Usage:
```c
#include "include/is_even.h"
#include <stdio.h>

int main(void) {
    printf("%d\n", is_even(4)); // 1
    printf("%d\n", is_odd(3));  // 1
}
```

### Python (`.pyi`)

```sh
maturin develop --features python
```

Type stubs are in `python/is_even_wasm.pyi`. To auto-generate stubs, add
[pyo3-stub-gen](https://github.com/Jij-Inc/pyo3-stub-gen) to your dependencies.

Usage:
```python
import is_even_wasm

print(is_even_wasm.is_even(4))  # True
print(is_even_wasm.is_odd(3))   # True
```

## Run browser demo

Serve with any static HTTP server (module scripts require HTTP, not `file://`):

```sh
npx serve .
```

Open `http://localhost:3000/demo.html`.

## Run Rust tests

```sh
cargo test
```

## API

| Function | Signature | Description |
|----------|-----------|-------------|
| `is_even` | `(n: int/i32/number) → bool` | `true` if `n` is even |
| `is_odd` | `(n: int/i32/number) → bool` | `true` if `n` is odd |
