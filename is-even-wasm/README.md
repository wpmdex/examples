# is-even-wasm

Rust → WebAssembly example. Checks if a number is even or odd. Demonstrates `wasm-bindgen` + auto-generated TypeScript declarations.

## Structure

```
is-even-wasm/
├── Cargo.toml          # Rust package config
├── src/lib.rs          # Rust source
├── demo.html           # Browser demo
└── pkg/                # Generated by wasm-pack (do not edit)
    ├── is_even_wasm.d.ts
    ├── is_even_wasm.js
    ├── is_even_wasm_bg.wasm
    └── package.json
```

## Prerequisites

- [Rust](https://rustup.rs/)
- [wasm-pack](https://rustwasm.github.io/wasm-pack/installer/)

```sh
cargo install wasm-pack
```

## Build

```sh
wasm-pack build --target web
```

Outputs to `pkg/`. The `.d.ts` file is generated automatically.

## Usage

### Browser (ESM)

```html
<script type="module">
  import init, { is_even, is_odd } from "./pkg/is_even_wasm.js";

  await init();

  console.log(is_even(4)); // true
  console.log(is_odd(3));  // true
</script>
```

### TypeScript

```ts
import init, { is_even, is_odd } from "./pkg/is_even_wasm.js";

await init();

const n = 42;
console.log(`${n} is even: ${is_even(n)}`); // 42 is even: true
```

## Run Demo

Serve with any static HTTP server (module scripts require HTTP, not `file://`):

```sh
npx serve .
```

Then open `http://localhost:3000/demo.html`.

## API

| Function | Signature | Description |
|----------|-----------|-------------|
| `is_even` | `(n: number) => boolean` | Returns `true` if `n` is even |
| `is_odd` | `(n: number) => boolean` | Returns `true` if `n` is odd |
